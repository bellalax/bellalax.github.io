Scratch Game!

The development process of remixing the scratch platform was a particularly tedious task. With having to first examine the code and understand it to add new content without altering the wrong section of code. Furthermore, my partner and I added a few additions to the code which were the ability to kill the enemy(Giga Sentries) with the space-bar, to earn points, having three lives, changing the player’s controls from WASD to the arrow pad, changing the player’s costume, sound effects and adding levels. In addition, to the alterations to the original game my partner and I ran into a few challenges. One of these challenges was when my partner and I tried to allow the player the ability to kill the enemy. The difficulty we ran into was that we overlooked a simple command in the code which was the forever loop. Another challenge we ran into was how the player’s character wasn’t changing costumes when the player moved. The problem was that in the player’s character’s code that we had forgotten in the forever loop to add the set costume demand. While this remix had many challenges, although by collaborating the challenges were fixed fast and effectively. As well as working together some steps were done independently, for example, I worked on the character’s costumes and backgrounds as well as finding new code we could add into the game off of other scratch games.
